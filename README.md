# Magic 8-Ball

A web-based **Magic 8-Ball** app that lets users ask questions and receive mystical answers. The project demonstrates skills in HTML, CSS, JavaScript, and multilingual support.

## Features

- Interactive 8-Ball with shaking animation
- Input field for user questions
- Random answers displayed inside the 8-Ball
- Multilingual support: English, German, Russian
- Language toggle using flag icons
- Responsive design for different screen sizes

## Demo

You can view the live project on GitHub Pages:

## Technologies Used

- HTML5
- CSS3 (Flexbox, animations)
- JavaScript (ES6+)
- JSON for translations
- Git & GitHub for version control and hosting
